Gênero:

Ilustração 2D digital

Objetivo:

A ilustração foi criada para um trabalho de desenvolvimento de jogos web do meu curso técnico de programação de jogos digitais, que consistia no desenvolvimento de um jogo em JavaScript, com alguma temática que contemplasse todos os requisitos prévios para o jogo, como animações, interfaces, ilustrações, sistema de pontuações. Todas as Ilustrações do jogo, que tinha como temática conceitos da física como cargas elétricas e materias supercondutores, foram feitas por mim.

Desenvolvimento:

A ilustração foi desenvolvida por mim no software de ilustração 2D chamado Ibis Paint X.
